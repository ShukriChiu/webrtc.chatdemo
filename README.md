# webrtc.chatdemo


### Introduce
本程序是基于websocket和webrtc做的一个在线实时视频流聊天室，制作起因是由于计算机网络课程的一个课程加分Project，顺带可以用来参加一个比赛，然后基于github上的一些开源项目做出了自己的一些修改和整合，现在依然还在不断改进之中。

Update:  

最近会开始更新项目，使用最新版本的Rails 搭建一个围绕Sneaker的兴趣小组
